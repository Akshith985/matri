# **App Name**: BloomCare

## Core Features:

- User Authentication: Secure user authentication using Firebase Authentication with Login and Sign-up functionality.
- Pregnancy Phase Management: Global state management for tracking the user's current pregnancy phase (Pre, During, Post).
- User Data Storage: Cloud Firestore integration for storing and retrieving user data.
- Profile Setup Script: Trigger a 'user profile' setup script on the first login to initialize the user's health dashboard.
- Health Dashboard: Personalized dashboard to display user-specific health information.
- AI Health Suggestions: Suggest customized information and recommendations based on a users specific pregnancy phase by using a LLM as a tool to determine relevancy.
- Mobile Responsive UI: A mobile-responsive UI using React and Tailwind CSS, that works with web browsers.

## Style Guidelines:

- Primary color: Soft Lavender (#E6E6FA) to evoke feelings of calmness and care, reflecting the nurturing aspect of the app.
- Background color: Off-White (#F8F8FF), a very desaturated version of the primary color, which provides a clean and gentle backdrop.
- Accent color: Muted Blue (#A9B7DF) creates contrast while remaining harmonious, for highlights and actionable items.
- Body font: 'PT Sans' (sans-serif) provides a modern, readable base for all body text.
- Headline font: 'Alegreya' (serif) is used for headers, as a contrasting, yet accessible display font.
- Use simple, clear icons in a line art style to represent various features and data points.
- A clean, intuitive layout that adapts seamlessly to different screen sizes, with clear hierarchy and spacing.
- Subtle transitions and animations to provide feedback and enhance user experience, without being distracting.
